2011/11/24	get subdirectories from current path

*** ListDirectory.bat
* This batch script lists subdirectories -non recursive- from a given directory 
* (in this case, local directory of the batch file) and saves the results in a txt file on the same folder. 
* Output file is named directory.txt

*** ListDirectory.XLS.bat
* This batch script lists subdirectories -non recursive- from a given directory 
* (in this case, local directory of the batch file) and saves the results in a xls file file on the same folder. 
* Output file is named directory.xls
